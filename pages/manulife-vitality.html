Manulife Vitality Savings Tool

Designer at Rich Media

A redesign of an existing tool that allowed users to know what rewards they could get if they joined the program. Included in the project scope was a dynamic PDF and video that compiled the user's choices and results into a personalized element.

tools used

Illustrator, Animate, VS Code

time frame

3 months

problem

Manulife Vitality promotes healthy living by rewarding customers who make healthy living choices. Based off an existing tool, Manulife wants a tool that informs users of how many points and what rewards they can get through this program.

How can we redesign the Vitality tool for Manulife?

goals

  1. Propose redesign that follows brand standards, project requirements and accessibility standards
  2. Use microanimations to show growth of points/rewards
  3. Execute UI/UX designs using HTML, CSS, and JS
  4. Animate an illustrated personalized results video
  5. Design a personalized PDF of user's results
  6. Allow for a dual UX flow*
  7. Ensure tool is responsive

As Lead Designer, I worked closely with Manulife to create a refreshing design that matches their branding. My team and I focused on enhancing the UI experience through microanimations and maintaining colorful elements.

*User can enter tool from multiple points
